Коротко опишите, какие задачи с помощью механизма ветвления решаются в GIT? Зачем выполняется их слияние?
Используя ветвление, можно вести независимую от основной версии линию разработки. Для объединения изменений выполняют слияние.

(2) Укажите команды для:

переименования веток;
$ git branch --move old-name new-name


создания и переключения на новую ветку;
git checkout -b <newbranchname>


удаление ветки;
git branch <branchname> -d

слияния веток.
git merge <newbranchname>
