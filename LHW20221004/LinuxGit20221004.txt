Linux git - 04.10.2022

First level: Укажите след. команды:

для полной очистки индекса
git rm --cached имя файла - убрать файл из репозитория
git rm -r --cached . - стереть весь кэш репозитория


для просмотра разницы между раб.каталогом и посл. коммитом    git diff --cached


для создания новой ветки и одновременного переключения в на нее git checkout -b conf

для сравнения веток между собой - git diff branch1 branch2

для удаления локальной ветки - git branch -d другая_ветка

для слияния веток - git merge другая_ветка

для перезаписи посл.коммита - git commit —amend
